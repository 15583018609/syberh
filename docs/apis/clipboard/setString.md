## syber.clipboard.setString(Object object)
向系统剪贴板设置内容

### 参数

**Object object**

| 属性 | 类型   | 是否必填 | 描述                   |
| ---- | ------ | -------- | ---------------------- |
| text | String | 是       | 向系统剪贴板设置的内容 |

### 示例
```js
syber.clipboard.setString({
    text:"some of the content"
});
```
