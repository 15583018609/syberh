## syberh.camera.captureVideoStart（Object object）


开始摄像功能



### **参数**

#### Object object

| 属性   | 类型   | 是否必填 | 描述                                      |
| ------ | ------ | -------- | ----------------------------------------- |
| camera | number | 是        | 指定摄像头，`1` ：主摄像头，`2`：辅摄像头 |



### 示例代码

```
syberh.camera.captureVideoStart({
	camera: '1'
});
```
